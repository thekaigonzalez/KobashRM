NAME

    terminals - Terminal Management from terminal-superuser

SYNOPSIS

    terminals
        Write a terminal using an absolute path.
        You can use this or use tkrun manually.

DESCRIPTION

    /etc/terminals is a txt file containing the initial terminal for klibc.

    /etc/terminals contains basic find-and-replace syntax:

        /: ./kobash-mac (HOME)

    if you're making custom distributions, you must change this to your distribution's ABS-PATH or else it'll Default to ./
    please see FOR DEVELOPERS section for more details.

FOR DEVELOPERS

    Please edit these files.

        ./kobash/0/kobash-lib.lua (line 72)
            local abc = string.gsub(terminal_to_run, "/", "./kobash-mac/", 1)
                                                            ^^^^^^^^^^^
            Change ./kobash-mac to your HOME directory (kobash-mac as HOME)
